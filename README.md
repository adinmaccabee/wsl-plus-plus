# WSL++

WSL++ is a minimal WinForms wrapper for Windows Subsystem for Linux.

## Features
- Tabbed WSL sessions
- Native WSL execution (no emulation)
- Simple menu bar (File / Edit / View)
- Automatic WSL detection
- Optional WSL installation prompt
- Multi-distro support

## Requirements
- Windows 11
- WSL installed (`wsl --install` if needed)
- At least one Linux distribution installed

## Run

```bash
dotnet build
dotnet run
